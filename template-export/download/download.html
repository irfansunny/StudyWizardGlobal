Getting Started

After downloading the template, follow these steps to get up and running:

  1. Extract the ZIP file to your preferred location
  2. Navigate to the template directory
  3. Run the setup script:
    ./setup.sh
  4. Start the development server:
    npm run dev
  5. Open your browser and visit:
    http://localhost:5000

Customization

This template is designed to be easily customizable:

  • Replace the logo and branding with your own
  • Update the color scheme via theme.json
  • Modify content for your specific business needs
  • Customize forms to collect the information you need
  • Add or remove destinations based on your offerings

For detailed customization instructions, refer to the CUSTOMIZATION.md file included in the download.

Technical Requirements

  • Node.js v14 or higher
  • NPM or Yarn package manager
  • Modern web browser
